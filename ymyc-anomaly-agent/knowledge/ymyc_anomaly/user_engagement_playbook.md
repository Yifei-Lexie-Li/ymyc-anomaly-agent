# User Engagement Playbook

## Returning Users

When completion or retention declines among returning users, investigate
whether the product is continuing to serve their established interests.
Repeated exposure to similar topics, weaker personalization, or an abrupt
change in content mix may reduce engagement.

Recommended checks:

- Compare topic consumption with each user's historical interests.
- Check whether returning users are receiving repetitive recommendations.
- Compare visit frequency and session depth with the previous period.
- Review whether high-performing recurring formats were reduced.

Possible actions:

- Improve recommendation personalization.
- Create re-engagement collections based on prior interests.
- Restore proven recurring content formats for established users.
